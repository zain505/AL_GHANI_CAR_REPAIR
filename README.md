# Al-Ghani (Replica)

This is a React (Vite) replica of the public landing page at `Al-Ghani-template.framer.website`, built with **pure CSS** (no Tailwind / no CSS-in-JS).

## Tech stack
- React (.jsx)
- HTML
- CSS (pure CSS, responsive via media queries only)
- JavaScript

## Run locally

1) Install dependencies
```bash
npm install
```

2) Start dev server
```bash
npm run dev
```

3) Production build
```bash
npm run build
npm run preview
```

## Notes
- Images are placed in `public/assets/`.
- The layout is mobile responsive using only CSS media queries.
